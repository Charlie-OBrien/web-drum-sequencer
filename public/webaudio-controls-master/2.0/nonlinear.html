webaudio-controls sample (NonLinear)

This is a non-linear value knob sample.

Here the `conv` attribute is conv="(x)=>{return Math.pow(10000,x/100)}", it means the knob's raw value range 0-100 is converted to exponential 1-10000.

To get converted value, use `knob.convValue` instead of `knob.value`.


<webaudio-knob conv="(x)=>{return Math.pow(10000,x/100)}"></webaudio-knob>
Raw Value : knob.value = 0
Converted Value : knob.convValue = 1
